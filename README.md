# fvtt-character-md-export

Модуль для FoundryVTT — экспортирует лист персонажа D&D 5e в Markdown,
оптимизированный для индексации в RAG-системе **Mercer**.

## Установка

### Через Git (рекомендуется)

```bash
cd /path/to/FoundryVTT/Data/modules
git clone https://github.com/Annener/fvtt-character-md-export.git
```

Обновление:
```bash
cd fvtt-character-md-export
git pull
```

### Вручную

Скачай архив и распакуй в `FoundryVTT/Data/modules/fvtt-character-md-export/`.

## Использование

1. Активируй модуль: **Настройки → Модули → Управление модулями**
2. Открой лист персонажа
3. Нажми кнопку **«MD Экспорт»** в шапке листа
4. Файл `ИмяПерсонажа.md` скачается автоматически

## Структура экспортируемого файла

```
# Имя персонажа
## Общие сведения
## Боевые характеристики
## Характеристики
### Спасброски
### Навыки
## Особенности расы
## Особенности класса
## Прочие черты и способности
## Инвентарь
## Заклинания
## Личность
## Биография
```

Каждый раздел — отдельный `##`-чанк, что позволяет RAG корректно
разбивать документ по смысловым блокам.

## Совместимость

- FoundryVTT 11+
- Система dnd5e (официальная)
