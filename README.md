# 📄 Character MD Export — FoundryVTT Module

> Модуль для **FoundryVTT**, который экспортирует лист персонажа D&D 5e в формат **Markdown**.  
> Идеально подходит для хранения, индексации и использования в RAG-системах и личных базах знаний.

---

## ✨ Возможности

- 📥 Экспорт полного листа персонажа одним кликом
- 📝 Структурированный Markdown с семантическими блоками (`##`-чанки)
- 🤖 Оптимизирован для индексации в RAG-системах (каждый раздел — отдельный смысловой блок)
- ⚔️ Поддержка официальной системы **dnd5e**
- 🔄 Совместим с FoundryVTT 12–14

---

## 📦 Установка

### Способ 1 — через интерфейс FoundryVTT (рекомендуется)

1. Откройте **Setup → Add-on Modules → Install Module**
2. В поле **Manifest URL** вставьте:
   ```
   https://raw.githubusercontent.com/Annener/fvtt-character-md-export/main/module.json
   ```
3. Нажмите **Install** и дождитесь завершения установки

### Способ 2 — через Git

```bash
cd /path/to/FoundryVTT/Data/modules
git clone https://github.com/Annener/fvtt-character-md-export.git
```

Обновление до последней версии:
```bash
cd fvtt-character-md-export
git pull
```

### Способ 3 — вручную

1. Скачайте [последний релиз](https://github.com/Annener/fvtt-character-md-export/releases/latest) в виде ZIP-архива
2. Распакуйте в папку `FoundryVTT/Data/modules/fvtt-character-md-export/`
3. Перезапустите сервер FoundryVTT

---

## ⚙️ Активация модуля в мире

1. Войдите в нужный **World** (игровой мир)
2. Откройте **Настройки** (шестерёнка) → **Управление модулями**
3. Найдите **Character MD Export** в списке и поставьте галочку
4. Нажмите **Save Module Settings**

---

## 🚀 Использование

1. Откройте **лист персонажа** (Actor Sheet) нужного персонажа
2. В шапке листа нажмите кнопку **«MD Экспорт»**
3. Файл `ИмяПерсонажа.md` автоматически скачается в папку загрузок

---

## 📂 Структура экспортируемого файла

Каждый раздел экспортируется как отдельный `##`-заголовок, что позволяет RAG-системам корректно разбивать документ по смысловым блокам:

```markdown
# Имя персонажа

## Общие сведения
## Боевые характеристики
## Характеристики
### Спасброски
### Навыки
## Особенности расы
## Особенности класса
## Прочие черты и способности
## Инвентарь
## Заклинания
## Личность
## Биография
```
